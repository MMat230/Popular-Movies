# PopularMovies

### Description
This application help users discover popular and recent movies using [TMDb API](https://www.themoviedb.org/documentation/api).

### Installation
- Obtain an TMDb API Key.
- In `TMDbApi.java`, change the `API_KEY` constant with your own.
- Build & run, enjoy.
